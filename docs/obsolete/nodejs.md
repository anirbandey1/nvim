# Nodejs provider


#### Nodejs provider

For Arch 
```
sudo pacman -S nodejs
sudo pacman -S npm
sudo npm -g update
sudo npm i -g yarn
sudo npm i -g neovim
```

For Ubuntu
```
curl -fsSL https://deb.nodesource.com/setup_19.x | sudo -E bash -

sudo apt-get install nodejs

# If possible update npm

# Install yarn
sudo npm i -g yarn
sudo npm i -g neovim
```

For Windows
```
# Run as administrator
choco install nodejs
choco install yarn
```
